Advanced RAG is a stronger retrieval augmented generation pattern built from the original naive RAG project.

In the local workflow, documents are loaded from the docs folder, split into sentence-aware overlapping chunks, ranked with query expansion, BM25-style lexical scoring, TF-IDF cosine semantic scoring, reciprocal-rank hybrid fusion, phrase boosts, source labels, and MMR diversity reranking, then passed as context to a Groq chat model.

The Vercel deployment uses the same advanced lightweight retrieval pattern before passing context to Groq.
